# ❌ Tic-Tac-Toe Game ⭕

An interactive, responsive, and clean web-based **Tic-Tac-Toe** game built using vanilla frontend technologies (HTML, CSS, and JavaScript). 

## 🚀 Features
* **Modern UI:** Clean layout with a minimalist design.
* **Real-time Status:** Displays turn indicators for Player X and Player O.
* **Win/Draw Checker:** Automatically detects and highlights winning combinations or declares a tie.
* **Reset Functionality:** Restart the game at any point with a single click without refreshing the browser.
* **Responsive Design:** Optimized for both desktop monitors and mobile screens.
